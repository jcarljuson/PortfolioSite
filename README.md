# Jcarl Juson — Portfolio

Clean, responsive portfolio showcasing projects in AI, Robotics, and Neurotechnology.

## Preview Locally

Serve the static files from the project root:

```bash
python -m http.server 5500
```

Open `http://localhost:5500/`.

## Structure

- `index.html` — main page
- `styles.css` — site styles
- `script.js` — small interactions (mobile nav, smooth scroll, About toggle)
- `profile.png`, `prosthetic.png`, `Award.png` — images
- `assets/` — fallback placeholder and notes

## Deploy

Push to GitHub and deploy with Vercel (no build step; static site).
